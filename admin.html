Supabase uchun SQL

Quyidagi SQL yangi qo'shgan rasmlaringizni katalogga yozadi. Eski rasmlarga tegmaydi.

1. Pastdagi SQL'ni Nusxa tugmasi bilan nusxalang.
2. Supabase → SQL Editor → yangi (toza) tab → qo'ying → Run.
3. Katalogni oching, Ctrl+Shift+R — rasmlar chiqadi.
